# ActivityNest

**Explore · Plan · Thrive**

ActivityNest is a family activity finder for camps, classes, sports, and events across Canada.

## Project Structure

- `index.html` — Main HTML page
- `styles.css` — All CSS styles
- `script.js` — Application logic and mock data

## Getting Started

Open `index.html` in a browser to view the prototype.

## Features

- Smart search with natural language queries
- Filter by age, location, season, budget, and radius
- Side-by-side activity comparison
- Budget planner with weekly calendar
- Interest-based category browsing
